# Fraud Detection Prediction App

A **Streamlit web app** that predicts whether a financial transaction could be fraudulent using a trained **machine learning pipeline**. Enter transaction details and get real-time predictions with clear alerts.

---

## Features

- User-friendly interface to input transaction details.  
- Predicts fraud in real-time using a pre-trained ML model.  
- Highlights suspicious transactions with a warning message.  
- Works directly in the browser with Streamlit.

---

## Technologies Used

- **Python 3.11**  
- **Streamlit** – for the interactive web app  
- **Scikit-learn** – for the ML model pipeline  
- **Pandas & NumPy** – for data handling  
- **Joblib** – to load the pre-trained model

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Pooja389/Fraud-detection
cd Fraud-detection
```
2. run this in terminal
```bash
streamlit run fraud_detect.py
```
